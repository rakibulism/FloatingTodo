[05] FAQ

Good questions.

Is it really free?

Yes. Today is free and open source — no account, no subscription, no trackers. Read every line on GitHub.

Does it work offline?

Completely. The native apps and the installable web app store your tasks locally on your device — there's no server and nothing to sign in to.

Why can't the web version float on top?

Browsers can't draw over other apps or launch themselves on a schedule — that's an OS limitation. So the web app is a clean, installable to-do list and uses notifications; the floating-on-top behavior lives in the native macOS & Windows apps.

How do the scheduled reminders work?

Each platform uses its native scheduler: launchd on macOS, Task Scheduler on Windows, and scheduled notifications on Android. You pick the times in Settings.

Does it update automatically?

Every app shows an in-app “update available” notice when a new version ships, so you never have to check back. The web app updates itself, and the Windows app installs updates silently on launch.
